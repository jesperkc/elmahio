Secure

Your log messages are shipped off to elmah.io through a 256 bit encrypted SSL connection. We do what is required for you to be able to sleep at night, keeping your log messages personal.

Unlimited Logs

All plans include unlimited number of logs. Split your logs by application, customer, date, you name it. We never limit the number of logs you need to run your business.

Visual Studio Integration

Elmah.io is the only cloud logging framework which integrates directly into Visual Studio. Browse your elmah.io logs from a window inside your favorite IDE.

Extending the standard

Elmah.io is based on ELMAH, the market standard for logging errors from any .NET web application. But we don’t stop there. Everything from verbose messages to fatal errors are supported by our .NET client and logging framework integrations.

Group similar messages to spot trends in your log data.

Elmah.io provides grouping by common properties like title, user, IP address, status code and much more.

Burst protection

Experience short periods of time with thousands of messages?

Don't worry, elmah.io offers an intelligent burst protection which only logs a certain amount of messages per minute.

Notifications

Receive an email every time a new message is logged.

Extra goodies

User management

Configure individual user access and authorization levels to your logs. You control the level of access, making it possible for people in your organization or even outside, to monitor your logs.

Invoicing

Make your accountant a happy camper with the invoicing feature. Invoices may not be a very sexy feature, but it is necessary to document how you spend the money in your business.

Hide messages

Hide messages you no longer want to look at or setup automatic rules, hiding every message logged, matching a specified query. Hidden messages can be toggled on and off through the search UI.

Extended user details

With the Extended User Details, you will be able to learn more about the users experiencing problems on your website. Browse their social media profiles or send them a mail directly from elmah.io

Mobile, tablet, desktop and up

Our dashboards are optimized to give a great overview of your application state. You can even use the Kiosk Mode to put the dashboard on a large flat screen.
